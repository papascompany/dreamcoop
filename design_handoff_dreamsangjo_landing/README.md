# Handoff: 드림상조협동조합 랜딩페이지

## Overview
대한민국의 비영리 상조 협동조합 "드림상조협동조합"의 단일 랜딩페이지 디자인입니다.
기존 상조회사가 보험·금융상품화된 것과 달리, "고인을 정중하고 고결하게 배웅하고
가족이 따뜻한 추억을 기억하도록 돕는다"는 철학을 전달합니다. 실사 사진 중심의
에디토리얼 레이아웃 + 스크롤에 반응하는 인터랙티브한 상조 진행 절차가 핵심입니다.

## About the Design Files
이 번들의 `.dc.html` 파일은 **HTML로 제작된 디자인 레퍼런스(프로토타입)**입니다.
의도한 룩앤필과 동작을 보여주는 참고물이며, 그대로 복사해 배포할 production 코드가
아닙니다. 작업의 목표는 이 디자인을 **대상 코드베이스의 기존 환경(React, Next.js, Vue 등)과
패턴/라이브러리에 맞게 재구현**하는 것입니다. 아직 환경이 없다면 프로젝트에 가장 적합한
프레임워크를 선택해 구현하면 됩니다.

> 파일 형식 참고: `.dc.html`은 이 디자인 도구의 "Design Component" 포맷(스트리밍 템플릿 +
> 로직 클래스)입니다. **브라우저에서 직접 열면 완성된 프로토타입이 그대로 동작**하므로
> 시각/인터랙션 레퍼런스로 활용하세요. 단, 이 포맷 자체를 그대로 가져갈 필요는 없고,
> 아래 명세를 기준으로 일반 컴포넌트로 옮기면 됩니다.

## Fidelity
**High-fidelity (hifi)**. 최종 컬러·타이포·간격·인터랙션이 모두 확정된 픽셀 단위 목업입니다.
대상 코드베이스의 기존 라이브러리/패턴을 사용하되, 아래 명세대로 픽셀에 가깝게 재현하세요.

## Interactions & Behavior
- **Sticky 헤더 전환**: `window.scrollY > 40`에서 solid 클래스 토글(배경/패딩/색상 0.5s 트랜지션).
- **진행 절차 스크롤 구동**: 타임라인 컨테이너의 `getBoundingClientRect()`로 진행도 계산
  → `prog = clamp((vh*0.6 - rect.top) / (rect.height - vh*0.2), 0, 1)`.
  fill 높이 = `prog*100%`, 각 노드 active 임계값 = `prog >= i/(n-1) - 0.06`.
  스크롤 이벤트는 `requestAnimationFrame`으로 스로틀, passive 리스너.
- **등장 애니메이션**: CSS `animation-timeline: view()` 기반(JS 클래스 토글 아님). 미지원 시 항상 표시.
- **FAQ 아코디언**: 단일 오픈 상태(state 인덱스), 클릭 토글.
- **서비스 카드 hover**: lift + shadow.
- **폼 제출**: `preventDefault` 후 완료 상태로 스왑(현재는 데모 — 실제 백엔드/전송 연동 필요).
- **반응형**: 900px 이하에서 네비 링크 숨김, 2단 그리드→1단, 서비스 4→2열, 통계 4→2열,
  타임라인은 좌측 정렬 단일 컬럼(스파인/노드 좌측 14px로 이동), 폼 2단→1단.

## State Management
- `scrolled`(헤더), `prog`(타임라인 진행도 0~1), `faq`(열린 인덱스, -1=닫힘), `sent`(폼 제출 여부).
- 데이터(STEPS·SERVICES·POINTS·STATS·FAQ)는 정적 배열. 실제 구현 시 CMS/상수로 분리 권장.
- 데이터 페칭 없음(폼 전송만 실제 API 연동 대상).

## Assets
- **사진**: 모든 이미지는 드래그&드롭 슬롯(placeholder)으로 비어 있습니다. 실제 실사 사진으로
  교체 필요 — 히어로(새벽/자연), 철학(가족·손), 서비스 4종, 명언 밴드, 진행절차 6단계, 협동조합.
  권장 톤: 따뜻하고 차분한 자연·인물 사진. (현재 프로토타입은 `image-slot.js` 컴포넌트로
  슬롯을 구현 — 재구현 시에는 일반 `<img>`/Next.js `<Image>`로 대체)
- **아이콘**: 로고 마크는 인라인 SVG(잎/촛불 형상). 그 외 별도 아이콘 없음(텍스트/SVG 위주).
- **폰트**: Google Fonts(위 import).
- **플레이스홀더 텍스트/수치**: 조합원 수, 8,400회, 98%, 전화 1668-0000, 주소·사업자번호는
  모두 **임시값** — 실제 정보로 교체.
